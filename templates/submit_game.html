{# 提交指南 #}

提交指南

  • 标题：2-100 字符，避免重复用名
  • 封面图：16:9 比例，≤ 10MB
  • 宣传图：最多 15 张，每张 ≤ 3MB
  • ZIP 包：≤ 500MB，必须包含 index.html
  • Python：需提供主文件与启动命令
{# 格式要求 #}

格式要求

  • 封面格式：JPG / PNG / GIF / WEBP
  • 截图格式：JPG / PNG / WEBP
  • 游戏文件：单 HTML 或 ZIP
  • Python 文件：.py（含依赖说明）
  • 建议在 ZIP 内放置 README 说明
{# 审核流程 #}

审核流程

1 提交：填写完成后点击「提交申请」
2 HTML / 单HTML：直接上线，无需审核
3 Python：进入审核队列，1-3 个工作日内反馈
4 更新：版本号递增即可提交新版本
