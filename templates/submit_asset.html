{# ============== 右侧：帮助侧栏 ============== #}
{# 提交指南 #}

素材提交指南

  • 标题：2-100 字符，避免与已有素材重名
  • 一句话卖点：突出核心价值，吸引点击
  • 封面图：16:9 比例，≤ 10MB
  • 预览图：3-10 张，每张 ≤ 10MB
  • 素材包：≤ 4GB，建议附带 README
{# 文件格式要求 #}

文件格式要求

  • 封面 / 预览图：JPG / PNG / GIF / WEBP
  • 素材包：.zip / .rar / .7z / .unitypackage
  • 封面大小：≤ 10MB
  • 预览图大小：每张 ≤ 10MB
  • 素材包大小：≤ 4096MB（4GB）
  • 预览图数量：3 - 10 张
{# 审核流程 #}

审核流程

1 提交：填写完成后点击「提交素材」
2 审核：管理员审核内容合规性与文件完整性
3 上架：审核通过后素材将对所有用户可见
{# 常见问题 FAQ #}

常见问题

支持 .zip、.rar、.7z、.unitypackage 四种主流打包格式，最大 4GB。建议在压缩包内放置 README 说明文件。
至少 3 张、最多 10 张预览图，每张不超过 10MB。建议 4:3 比例，清晰展示素材内容。
支持多选：可商用（允许商业用途）、需署名（使用时需注明作者）、禁止二次分发（不得再次分发或转售）。
一般 1-3 个工作日内完成审核。付费素材或可疑内容可能延长。审核结果将通过通知告知。
